새로운 프로젝트 폴더인 `sq-test/` 구조에 맞추어 2탄 [SQ 멘탈 웰니스 및 스트레스 테스트]의 프론트엔드를 독립적으로 구축해 줘. 파일명은 폴더가 분리되었으므로 `sq_` 접두사를 빼고 깔끔하게 처리할 거야.

1. 📂 프로젝트 폴더 및 파일 구조 안내
- `sq-test/index.html` (메인 시작 화면 및 BGM 토글 버튼)
- `sq-test/questions.html` (질문 진행 화면 및 프로그레스 바)
- `sq-test/analysis.html` (8대 테마 상세 결과 화면 및 성향 도감)
- `sq-test/questions.json` (제공된 20개 정밀 문항 데이터)
- `sq-test/analysis.json` (제공된 16가지 성향별 8대 테마 분석 데이터)
- `assets/bgm.mp3` (배경음악 파일 - 상위 또는 공통 폴더 경로에 맞춰 연결)

2. 🛠️ 자바스크립트 및 데이터 연동 요구사항
- `questions.html`에서 같은 폴더 내의 `./questions.json`을 Fetch로 읽어와 화면에 순차적으로 렌더링해 줘.
- 유저가 선택지를 누를 때마다 각 지표(R, E, T, V)의 score가 누적 합산되도록 스코어링 엔진을 구현해 줘.
- 20번 문항이 끝나면 3초간의 "정밀 분석 중... 대기 화면"을 거친 뒤, 계산된 최종 유형 코드(예: INFP)를 가지고 결과 페이지로 이동시켜 줘. (이동 주소 형식: `./analysis.html?type=INFP`)
- `analysis.html`에서는 URL 파라미터로 넘어온 `type` 값을 인식하고, 같은 폴더 내의 `./analysis.json`에서 해당 유형의 데이터를 매핑하여 8개 그리드 카드(🔋, 🧘, ✨, 💥, 📿, 🚨, 💸, 💊)에 예쁘게 뿌려줘.

3. 🎨 UI/UX 디자인 및 부가 기능
- 레이아웃 가이드: [데스크톱 화면에서는 최대 너비 600px 제한 + 중앙 정렬 앱 스타일 / 모바일 화면에서는 100% 꽉 찬 가로 1열 반응형] 디자인을 완벽히 적용해 줘.
- 질문지 하단, 대기 화면 정중앙, 결과지 하단에 구글 애드센스 광고가 들어갈 수 있도록 미리 `class="adsense-slot"`을 가진 그레이 톤의 깔끔한 플레이스홀더 영역을 레이아웃에 배치해 줘.
- 배경음악 ON/OFF 토글, 클릭 효과음, [성향 도감 모달 전체 구경하기], [현재 보고 있는 결과지 상세 URL 전체 복사] 기능까지 완벽하게 작동하도록 마감 코딩을 완료해 줘.
